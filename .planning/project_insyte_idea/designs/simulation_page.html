How does a Hash Table work?

ELI5 Expert

What is a Hash Table?

A hash table is a data structure that maps keys to values for highly efficient lookup. Think of it like a massive digital library where instead of searching row by row, you have a magic index that tells you the exact shelf and position of any book instantly.

lightbulb Try typing a key in the input box →

What happens on insert?

  1. 1 The key (e.g., "alice") is passed into a Hash Function.
  2. 2 The function returns an integer called the hash code.
  3. 3 This code is mapped to an index in the array (e.g., index 3).

"Notice how the same key always produces the same hash. This determinism is the secret sauce of O(1) complexity."

What about collisions?

Sometimes two different keys map to the same bucket. This is called a collision. We handle these using strategies like Chaining or Open Addressing.
